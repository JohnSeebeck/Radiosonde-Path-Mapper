# Radiosonde-Path-Mapper
A basic python script that takes in raw radiosonde data from an archived sounding, and outputs a 3d matplotlib graph showing the path of the sounding through 3d space.

Retrieve raw data in csv format, and edit the second-to-last line to master_function("[name of your csv file here].csv", "medium") by way of any text editor to use.

The script is only designed for use with radiosonde data, and its functionality with unrelated csv files is not guaranteed.
